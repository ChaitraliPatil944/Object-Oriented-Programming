# Object-Oriented Programming Handbook

This repository contains a structured handbook for learning Object-Oriented Programming (OOP) with Java.

## Overview

The content is organized into parts covering the foundations of OOP, core concepts, advanced design principles, Java language features, generics, exception handling, persistence, concurrency, SOLID principles, design patterns, enterprise application concepts, and interview-focused JVM topics.

## Repository Structure

The handbook is divided into the following sections:

- Part 1 — Introduction to Object-Oriented Programming
- Part 2 — Evolution of OOP
- Part 3 — Objects, Classes and Memory
- Part 4 — JVM Memory Model for OOP
- Part 5 — Encapsulation
- Part 6 — Abstraction
- Part 7 — Inheritance
- Part 8 — Polymorphism
- Part 9 — Relationships Between Objects
- Part 10 — Advanced Object Design
- Part 11 — Java Language Features Supporting OOP
- Part 12 — Generics and Type System
- Part 13 — Exception Handling in OOP
- Part 14 — Object Persistence
- Part 15 — Concurrency in OOP
- Part 16 — SOLID Principles
- Part 17 — Design Patterns
- Part 18 — OOP in Enterprise Applications
- Part 19 — JVM Internals for OOP Interviews
- Part 20 — Interview Mastery

## Purpose

This handbook is intended for:

- Students learning OOP concepts
- Java developers revising core principles
- Interview preparation for OOP and Java-related roles

## How to Use This Repository

Browse the lesson files in each part to study the topics in order. You can also use this repository as a reference guide while practicing Java programming.
